
# SOC Incident Dashboard

A full-stack cybersecurity dashboard project built with:

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- JSON APIs

## Features

- Incident dashboard
- Severity filtering
- Search functionality
- REST API
- JSON data storage
- Cybersecurity-themed UI

## Run the Backend

```bash
cd backend
npm install
node server.js
```

## Run the Frontend

Open `frontend/index.html` in your browser.

## API Endpoints

### GET incidents

```bash
GET /incidents
```

### POST incident

```bash
POST /incidents
```
